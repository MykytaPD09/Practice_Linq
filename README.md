# Practice_Linq


## Опис

Цей проект містить приклади використання LINQ (Language-Integrated Query) в мові програмування C# для обробки та аналізу даних з футбольних матчів.

## Використання

1. Запустіть програму та переконайтеся, що ви вказали правильний шлях до файлу "results_2010.json" у змінній `path`.

2. Викликайте методи `Query1`, `Query2`, ..., `Query10`, щоб виконати відповідні запити до даних.

3. Переглядайте результати в консолі.

## Висновок

Проект "Practice_Linq" є прикладом використання мови програмування C# та LINQ для обробки та аналізу даних з футбольних матчів. За допомогою бібліотеки Newtonsoft.Json здійснюється робота з JSON-файлом "results_2010.json", який містить інформацію про футбольні ігри.

У проекті реалізовані десять запитів (Query1 - Query10), кожен з яких виконує конкретну аналізу даних та виводить відповідні результати. Код відображає чітку структуру та ефективне використання можливостей LINQ для фільтрації та обробки інформації.
